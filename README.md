# AI News & Tools Aggregator v2.0

Aggregates AI news and tools from 10+ sources every 6 hours. Summaries, categories, and relevance scores generated by AI (Groq primary, Gemini fallback). Results published to a public dashboard on GitHub Pages.

## Dashboard

```
https://{your-username}.github.io/{repo-name}/
```

## Quick setup

### 1. Google Cloud — Service Account (10 min)
1. Go to [console.cloud.google.com](https://console.cloud.google.com) → new project.
2. Enable **Google Sheets API**.
3. IAM → Service Accounts → Create → download the JSON key.

### 2. Google Sheets (2 min)
1. Create a new sheet at [sheets.google.com](https://sheets.google.com).
2. Add two tabs: `AI_Feed` and `_meta`.
3. Row 1 of `AI_Feed`: `fecha | tipo | categoria | relevancia | nombre | link | precio | resumen | tags | fuente | idioma`
4. Share → "Anyone with the link can view" (public read).
5. Share → add the service account email as **Editor**.
6. Copy the Sheet ID from the URL (`/d/{ID}/edit`).

### 3. API Keys (5 min)
- **Groq**: [console.groq.com](https://console.groq.com) → API Keys → Create
- **Gemini**: [aistudio.google.com](https://aistudio.google.com) → Get API Key

### 4. GitHub Secrets (3 min)
Settings → Secrets → Actions → New secret:
- `GROQ_API_KEY`
- `GEMINI_API_KEY`
- `GOOGLE_CREDENTIALS_JSON` — full content of the service account JSON
- `SHEET_ID` — your Google Sheet ID

### 5. GitHub Pages (1 min)
Settings → Pages → Source: **Deploy from branch** → `main` → `/ (root)`.

### 6. Update dashboard
Open `index.html` and replace `YOUR_SHEET_ID_HERE` with your actual Sheet ID.

### 7. First run
Actions → "AI News Aggregator" → Run workflow → verify green.

## Adding/removing sources
Edit [`config/sources.yaml`](config/sources.yaml) and push. No Python code changes needed.

## Project structure

```
├── index.html                    # Public dashboard (GitHub Pages)
├── .github/workflows/
│   └── aggregator.yml            # Runs every 6 hours
├── src/
│   ├── main.py                   # Orchestrator
│   ├── collector.py              # RSS + HN fetcher
│   ├── enricher.py               # Full article text extraction
│   ├── dedup.py                  # SHA256 deduplication
│   ├── ai_enricher.py            # Groq/Gemini AI enrichment
│   └── sheets_writer.py          # Google Sheets writer + retention
├── config/
│   └── sources.yaml              # Data sources (no code needed)
├── data/
│   └── hashes.json               # Processed URL hashes
└── requirements.txt
```
